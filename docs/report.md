# Отчёт — Задание 3. Телефонная книга CONTACT (Вариант 37)

## Описание задачи

Описать структуру CONTACT с полями:
- Name — фамилия и инициалы
- TELE — номер телефона
- DATE — дата рождения (год, месяц, число)

Программа выполняет:
- Ввод с клавиатуры данных в массив BLOCKNOTE из 10 структур типа CONTACT
- Записи упорядочены по возрастанию даты рождения
- Вывод сведений о человеке по номеру телефона
- Если такого человека нет — сообщение
